① Netflix Clone :: React 기반의 웹 구현

개인 프로젝트 : 참여도 100%

제작 기간 : 2일

사용툴 : Figma, HTML5, JAVASCRIPT, REACT(SCSS)

React를 활용해 Netflix의 핵심 기능을 재현한 클론 웹 어플리케이션을 개발했습니다.
UI/UX 디자인과 비동기 데이터 처리에 중점을 두었고, React, React Router, TMdb API를 연동하여 기능을 구현했습니다.
레이아웃은 Flex를 기반으로 구성했으며, Figma에서 디자인을 먼저 완성한 후 HTML, JavaScript, SCSS로 실제 구현했습니다.
